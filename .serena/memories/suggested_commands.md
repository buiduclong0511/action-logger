# Suggested Commands

## Development
This is a Chrome Extension with no build step. Development is done by editing JS/HTML/CSS files directly.

### Load Extension in Chrome
1. Open `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked" → select the project directory

### Reload After Changes
- Click the refresh icon on the extension card in `chrome://extensions/`
- Or use Ctrl+R on the extensions page

## System Commands (Darwin/macOS)
- `git` — version control
- `ls`, `find`, `grep` — file navigation and search
- No package manager needed (no dependencies)
- No build/test/lint commands (vanilla JS project)
